Quadtree based visual Compression.

load up an image, and select how to compress it.

Quadtree uses a quadtree to recusively devide the image into quadrants.

'Split by Color Variance' splits those quadrants depending on how much color varies within that region. This results in edges and regions of high color variance being found first and getting more pixels devoted to that region.

You can also set whether to use circles or squares, the size ratio (how large pixels are to thier orginal counterpart), and a Size limit that sets the pixel depth (so 100 size limit will be a image with 100 times less resolution)

Blockchop is for comparison

it compresses the image by cutting it into blocks of X*Y pixels and taking one pixel from that region. So block chop with a 1x2 bock will make an image with half the number of pixels horizontaly. There are also a few different rendering modes.

Quadtree Compression Visualizer

A Quadtree based compression algorithm that splits nodes by finding the node with the highest statistical variance of color.
